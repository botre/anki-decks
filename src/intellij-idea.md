# IntelliJ IDEA

## Add file to VCS

`⌥ + ⌘ + A`

## Toggle Find window

`⌘ + 3`

## Toggle Problems window

`⌘ + 6`

## Toggle Git window

`⌘ + 9`

## Toggle Terminal window

`⌥ + F12`

## Focus Terminal window

`⌥ + F12`

## Go back to editing file when Terminal is focused

`Esc`

## Navigate with the Select In popup

`⌥ + F1`
