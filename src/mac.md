# Mac

## ⌘

Command

## ⇧

Shift

## ⌥

Option/alt

## ⌃

Control

## Switch to the previous application (AltTab)

Press `⌘ + Tab`

## Open application switcher (AltTab)

Hold `⌘ + Tab`

## Close one application window

`⌘ + W`

## Close all application windows

`⌘ + ⌥ + W`

## Lock screen

`Control + ⌘ + Q`
