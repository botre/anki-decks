`⌘ + Tab`

`⌘ + Tab`

`Tab`

`Shift`

`W`

## Application switcher: hide selected window

`H`

`Q`

`⌘ + W`

`⌥ + ⌘ + W`

`⌘ + H`

## Quit an application

`⌘ + Q`

`⌘ + ,`

`Control + ⌘ + Q`
