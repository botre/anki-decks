# Linux

## Mark a file as executable

```bash
chmod +x filename.sh
```

## Bash shebang

```bash
#!/bin/bash
```

## Set of standard operating system interfaces based on the Unix operating system

POSIX

## Terminal: scroll up one page

`Shift + PageUp`

## Terminal: scroll down one page

`Shift + PageDown`

## FZF: search directory

Control + T

## FZF: search history

Control + R
