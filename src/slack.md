# Slack

## Open "Jump to"

`⌘ + K`

## Open "Direct messages"

`⌘ + Shift + K`

## Open "Threads"

`⌘ + Shift + T`

## Add a new line in chat

`Shift + ↵`

## Edit previous message

Focus message using `↑`, then press `e`.

## Command: toggle notifications

`/dnd`

## Command: toggle channel mute

`/mute`

## Command: list channel members

`/who`
